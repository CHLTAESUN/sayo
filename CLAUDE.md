# SAYO 공동 작업 안내

이 폴더는 Codex와 Claude가 함께 작업하는 SAYO 프로젝트입니다.

## 실행

```powershell
$env:Path='C:\Program Files\nodejs;'+$env:Path
npm run dev -- --host 127.0.0.1 --port 5173
```

현재 미리보기 주소: `http://127.0.0.1:5173`

## 프로젝트 성격

- 이름: SAYO (`Say + Open`)
- 대상: 한국 사용자를 위한 공개 대화 중심 커뮤니티
- 핵심: 짧은 글, 연결된 답글, 별, 재게시, 인용, 1:1 메시지
- 디자인: 밝은 민트 배경, 청록색 브랜드, 담백한 공개 대화 UI
- 모임 기능은 사용하지 않음

## 현재 구현

- React + Vite 단일 화면 프로토타입
- 게시글 작성, 사진 선택, 별, 답글 펼치기/접기, 재게시
- 인기 게시물 1~5위
- 우측 세로형 메시지 목록과 필요할 때만 열리는 대화창
- 단계형 회원가입과 로그인
- 완료된 인증 단계는 자동으로 접힘
- 가입 계정은 현재 로컬 테스트를 위해 `localStorage`에 저장

## 중요한 보안 사실

- 현재는 실제 서비스가 아닌 로컬 프로토타입이다.
- PASS 실명인증, 해외 IP 차단, 서버, 데이터베이스는 아직 연결되지 않았다.
- 실제 PASS 인증처럼 보이거나 보안이 적용됐다고 주장하지 않는다.
- 운영 구조와 요구사항은 `SECURITY.md`를 따른다.
- 주민등록번호 원문은 절대 수집하거나 저장하지 않는다.

## 공동 편집 규칙

- Codex도 같은 파일을 편집할 수 있으므로 수정 전에 최신 파일을 다시 읽는다.
- 사용자 요청 범위 밖의 기존 변경을 되돌리지 않는다.
- 주로 수정할 파일은 `src/main.jsx`, `src/styles.css`이다.
- 변경 후 `npm run build`를 실행한다.
- 현재 개발 서버를 중복 실행하지 않는다.

## 최근 사용자 결정

- 비밀번호: 8자 이상, 영문 포함, 숫자 포함, 확인값 일치
- 가입 완료 전 이메일 인증과 본인확인이 모두 필요
- 인증 완료 영역은 자동으로 접어서 한 줄 완료 카드로 표시
- 관심 주제 선택은 건너뛸 수 있음
- 회원가입 배경은 불투명
- 메신저 접속 상태는 표시하지 않음
